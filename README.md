# Getting Started

1. Clone this repo

```
$ git clone https://github.com/karuppiah7890/fancy-buttons.git
```

2. Install a web server to serve these files. I am using [`serve`](https://www.npmjs.com/package/serve) for this

```
$ npm i -g serve
```
